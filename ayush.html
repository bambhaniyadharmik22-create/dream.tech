API Endpoints

Search TM2 Codes
GET /api/tm2/search?q={query}&category={category}

Search for TM2 codes with optional category filtering.

Parameters:
  • q: Search query (e.g., "yoga", "ayurveda")
  • category: Optional category filter (diagnosis, treatment, medicine, procedure)
Example Response:
{
    "success": true,
    "data": [
        {
            "code": "TM2-YG001",
            "description": "Yoga Therapy for Back Pain",
            "category": "treatment",
            "icd11_equivalent": "MA82.Y",
            "namaste_code": "NAMASTE-YG-001"
        }
    ]
}
Map to ICD-11
GET /api/map/icd11/{tm2_code}

Get the ICD-11 equivalent for a TM2 code.

Example Response:
{
    "success": true,
    "data": {
        "tm2_code": "TM2-YG001",
        "icd11_code": "MA82.Y",
        "description": "Yoga therapy"
    }
}

System Architecture

Sample API Client Code (Python)
import requests

class TM2Client:
    def __init__(self, api_key):
        self.base_url = "https://api.tm2.in"
        self.headers = {
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        }
    
    def search_codes(self, query, category=None):
        params = {'q': query}
        if category:
            params['category'] = category
            
        response = requests.get(
            f"{self.base_url}/api/tm2/search",
            headers=self.headers,
            params=params
        )
        return response.json()

# Initialize client
client = TM2Client("your_api_key_here")
results = client.search_codes("yoga", "treatment")
print(results)
Sample API Client Code (JavaScript)
class TM2Client {
    constructor(apiKey) {
        this.baseUrl = "https://api.tm2.in";
        this.headers = {
            'Authorization': `Bearer ${apiKey}`,
            'Content-Type': 'application/json'
        };
    }
    
    async searchCodes(query, category = null) {
        const params = new URLSearchParams({ q: query });
        if (category) params.append('category', category);
        
        const response = await fetch(
            `${this.baseUrl}/api/tm2/search?${params}`,
            { headers: this.headers }
        );
        return await response.json();
    }
}

// Initialize client
const client = new TM2Client("your_api_key_here");
client.searchCodes("yoga", "treatment")
    .then(results => console.log(results));
